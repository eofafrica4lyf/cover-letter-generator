# Quick Start Guide

## Get Running in 3 Minutes

### 1. Install Dependencies (1 minute)
```bash
cd cover-letter-app
npm install
```

### 2. Start the App (30 seconds)
```bash
npm run dev
```

Open http://localhost:5173

### 3. Create Your First Cover Letter (90 seconds)

#### Step 1: Create Profile (30 seconds)
1. Click "Profile" in the navigation
2. Fill in your name, email, phone
3. Add at least one work experience OR education
4. Add a few skills
5. Click "Save Profile"

#### Step 2: Add a Job (30 seconds)
1. Click "Add Job" in the navigation
2. Choose "Paste Text" tab
3. Copy a job description from any job site
4. Paste it in the text area
5. Click "Parse Text"
6. Review the auto-filled information
7. Switch to "Manual Entry" tab
8. Fill in any missing fields (job title, company)
9. Click "Save Job Posting"

#### Step 3: Generate Cover Letter (30 seconds)
1. You'll be redirected to generation page
2. Review any information gaps (optional)
3. Select your preferred tone
4. Click "Generate Cover Letter"
5. Review your cover letter
6. Click "Edit & Export"

#### Step 4: Export (10 seconds)
1. Make any edits you want
2. Click "Export as PDF" (or DOCX/TXT)
3. Done! Your cover letter is downloaded

## Tips for Best Results

### Profile Tips
- Add 2-3 work experiences for professional roles
- Add academic context if you're a student
- Include 5-10 relevant skills
- Keep descriptions concise but specific

### Job Input Tips
- **Best method**: Paste the full job description text
- The system will auto-detect the language
- Review and correct any auto-filled information
- Add specific requirements if they're not captured

### Generation Tips
- Fill in recommended gaps for better quality
- Use "Professional" tone for most jobs
- Use "Enthusiastic" for startups/creative roles
- Use "Formal" for government/corporate positions

### Export Tips
- Edit before exporting for best results
- PDF is best for email submissions
- DOCX is best if you want to edit further
- TXT is good for copying into online forms

## Common Workflows

### Applying to Multiple Jobs
1. Create your profile once
2. Add all jobs you're interested in
3. Go to "My Jobs" to see all saved jobs
4. Click any job to generate a cover letter
5. Each letter is automatically saved
6. View all letters in "My Letters"

### Student/Internship Applications
1. In your profile, fill in "Academic Context"
2. Add your degree, university, expected graduation
3. Add relevant coursework
4. When adding jobs, select "Internship" or "Praktikum"
5. The system will emphasize your academic achievements

### International Applications
1. The system auto-detects job posting language
2. Your cover letter will be generated in that language
3. You can manually override if needed
4. Your profile stays in your language
5. The system handles the translation context
